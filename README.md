# AiAgentsChat
AiAgentsChat - мой личный проект - симуляция комнаты с Ai агентами.
## Суть проекта:
Пользователь создает какое-то количество Ai-Агентов и запускает их в общую комнату.
Пользователь задает тему для обсуждения (можно свободную тему),
А Ai агенты начинают развивать диалог между собой.
## Как запустить проект:
1) Скачайте LM Studio и запустите сервер с локальной нейронкой
2)Перейдите в режим разработчика:
   
 <img width="2559" height="1430" alt="Screenshot_3" src="https://github.com/user-attachments/assets/11f81fa1-8eec-446c-9b38-dbbebe46a184" />
4) Там вы сможете найти инфоромацию о модели и URL сервера. 
5) По умолчанию "http://localhost:1234/v1". Порт вы можете поменять в LM Studio, но не забульте его поменять и в приложении (временно поле ServerUrl находится в классе Constants)
6) Так-же в классе Constants есть поле ModelName = "openai/gpt-oss-20b". Если вы используете другую модель, скопируйте её из LM Studio и поменяйте поле ModelName.
7) Запускайте проект AiAgentsChat.Api. Через Swagger вы можете протестировать небольшой функционал, который на данный момент есть
8) А именно: создание Агентов и общение с ними по уникальному имени.
9) Дальше функционал будет расширяться, пока не получится полноценная симуляция команты с Агентами
