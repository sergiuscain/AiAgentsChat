# AiAgentsChat
AiAgentsChat - мой личный проект - симуляция комнаты с Ai агентами.
# Фронт здесь -> [Тык](https://github.com/sergiuscain/ai-chat-frontend)
## Суть проекта:
Пользователь создает какое-то количество Ai-Агентов и запускает их в общую комнату.
Пользователь задает тему для обсуждения (можно свободную тему),
А Ai агенты начинают развивать диалог между собой.
## Как запустить проект:
* Скачайте LM Studio и запустите сервер с локальной нейронкой 
* Перейдите в режим разработчика:
   
 <img width="2559" height="1430" alt="Screenshot_3" src="https://github.com/user-attachments/assets/11f81fa1-8eec-446c-9b38-dbbebe46a184" />
 
* Там вы сможете найти инфоромацию о модели и URL сервера. 
* По умолчанию "http://localhost:1234/v1". Порт вы можете поменять в LM Studio, но не забульте его поменять и в приложении (временно поле ServerUrl находится в классе Constants)
* Так-же в классе Constants есть поле ModelName = "openai/gpt-oss-20b". Если вы используете другую модель, скопируйте её из LM Studio и поменяйте поле ModelName.
* Запустите RabbitMq на LocalHost.
* Опционально: фронт на react -> [Тык](https://github.com/sergiuscain/ai-chat-frontend). Качаем, вводим Npm start и тестим через фронт,
* Запускайте проект AiAgentsChat.Api. Через Swagger вы можете протестировать небольшой функционал, который на данный момент есть
* А именно: создание Агентов и чатов. В чат можно добавить сколько угодно агентов, но при большом количестве агентов будет небольшой хаос в виде спама.
* Дальше функционал будет расширяться, пока не получится полноценная симуляция команты с Агентами
